The confirmation dialog service allows you to launch a simple confirmation dialog in a consistent way in SKY UX applications. The SkyConfirmationDialogService provider lauches the confirmation dialog. Within the SkyConfirmationDialogService service, you have options to display a description, type of dialog, and button text. Opens a confirmation dialog. The config parameter is an object literal to be passed to the component's constructor. This method returns the confirmation dialog instance. A string property of config that displays the message in the dialog. An enum of type SkyConfirmationDialogType used for specifying the number of buttons and default text. An optional array of SkyConfirmationDialogButton that allow you to override the default button config. A single button dialog with 'OK' as the default button text. Integer value equivalent is 1. A two button dialog with 'Yes' and 'Cancel' as the default button text. Integer value equivalent is 2. A three button dialog with 'Yes', 'No', and 'Cancel' as the default button text. Integer value equivalent is 3. A string property that displays the button text. A boolean property used to specify whether this button should be autofocused. This is the event that is emitted by the confirmation dialog instance when the dialog is closed. A string is passed back and has the text of the button that the user clicked.
