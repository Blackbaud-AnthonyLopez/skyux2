The confirmation dialog service allows you to launch a simple confirmation dialog in a consistent way in SKY UX applications. The SkyConfirmationDialogService provider lauches the confirmation dialog. Within the SkyConfirmationDialogService service, you have options to display a description, and specify the text of the confirm and cancel buttons. Opens a confirmation dialog. The config parameter is an object literal to be passed to the component's constructor. This method returns the modal instance. A string property of config that displays the text in the dialog. A string property of config that displays the confirm button text in the dialog. This has a default value of 'Yes'. A string property of config that displays the cancel button text in the dialog. This has a default value of 'Cancel'. This is the event that is emitted by the modal instance when the dialog is closed. The data property of the result object will be a string with value 'confirm' meaning the confirm button was clicked and 'cancel' meaning the cancel button was clicked.
